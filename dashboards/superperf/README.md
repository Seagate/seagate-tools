# SuperPerf

## Project setup
```shell
npm install
```
Update .env.development and .env.production files with "VUE_APP_REST_URI" as a host URL of REST Server.

### Compiles and hot-reloads for development
```shell
npm run serve
```

### Compiles and minifies for production
```shell
npm run build
```

### Lints and fixes files
```shell
npm run lint
```
